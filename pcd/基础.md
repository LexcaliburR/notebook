# 一. 三纬信息表达方式
1. mesh
   - 古早时期的游戏里面使用较多
2. voxel grid
   - 简单但不高效
3. octree
   - 效率比较高，只有在有物体的地方才会存在点
4. 点云 
   - 3d信息
   - 结构简单

# 二. 点云处理的难点
1. 稀疏性
2. 不规则
3. 缺乏纹理信息
4. 无序性
